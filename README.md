# Kindship Homebrew Tap

Official Homebrew tap for [Kindship CLI](https://kindship.ai/cli).

## Installation

```bash
brew tap kindship-ai/tap
brew install kindship
```

## Upgrade

```bash
brew upgrade kindship
```
